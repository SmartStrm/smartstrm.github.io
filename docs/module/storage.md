# 存储管理

SmartStrm 能够连接多种云端和本地存储。在正式开始创建生成任务前，你需要先在 **存储管理** 模块中配置好你的账号。

## 配置流程

1. 点击 **添加存储**。
2. **存储名称**：起一个好记的别名（如 `myquark`），该名称将用于任务引用
3. **选择驱动**：根据你的网盘类型选择对应驱动
4. **鉴权信息**：根据网盘驱动的不同，通过 **扫码登录** 或 **Cookie** 等方式授权

> [!TIP] 提示
> 建议存储配置保存后先点击 **测试** 按钮确认连接成功后再进行后续的任务配置

## 支持的驱动一览

| 驱动名称            | 说明                                                               |
| :------------------ | :----------------------------------------------------------------- |
| **OpenList**        | 可转接各类网盘、WebDAV等                                           |
| **WebDAV**          | 兼容所有标准 WebDAV 协议，支持 123 VIP、可转接 CloudDrive2         |
| **本地存储**        | 将已挂载的文件、或本地文件再次映射为 .strm                         |
| **夸克网盘**        | 支持原码代理和转码直链，如需使用转码直链(302)需要抓取移动端请求URL |
| **夸克网盘TV**      | 支持扫码登录，但接口仅支持只读访问（浏览和下载）                   |
| **115开放平台**     | 通过开放平台API调用，合规使用官方接口，建议设置访问限频            |
| **天翼云盘**        | 电信播放速度极快，支持302直链                                      |
| **123云盘开放平台** | 需开通开发者权益包方可调用                                         |
| **ANi Open**        | 一个支持直链的番剧公益站                                           |
