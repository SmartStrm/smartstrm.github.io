# 存储管理

SmartStrm 能够连接多种云端和本地存储。在正式开始创建生成任务前，你需要先在 **存储管理** 模块中配置好你的账号。

## 配置流程

1. 点击 **添加存储**。
2. **存储名称**：起一个好记的别名（如 `myquark`），该名称将用于任务引用
3. **选择驱动**：根据你的网盘类型选择对应驱动
4. **鉴权信息**：根据网盘驱动的不同，通过 **扫码登录** 或 **Cookie** 等方式授权

> [!TIP] 提示
> 建议存储配置保存后先点击 **测试** 按钮确认连接成功后再进行后续的任务配置

## 支持的驱动一览

| 驱动名称            | 说明                                                               |
| :------------------ | :----------------------------------------------------------------- |
| **OpenList**        | 可转接各类网盘、WebDAV、连接远程文件资源等                         |
| **WebDAV**          | 兼容所有标准 WebDAV 协议，支持 123 VIP、可转接 CloudDrive2         |
| **本地存储**        | 将已挂载的文件、或本地文件再次映射为 .strm                         |
| **夸克网盘**        | 支持原画代理和转码直链，如需使用转码直链(302)需要抓取移动端请求URL |
| **夸克网盘TV**      | 支持扫码登录，但接口仅支持只读访问（浏览和下载）                   |
| **115开放平台**     | 通过开放平台API调用，合规使用官方接口，建议设置访问限频            |
| **天翼云盘**        | 支持原画直链，支持家庭云，电信播放速度极快，支持 .cas 秒传播放     |
| **123云盘开放平台** | 2026年3月后，需开通开发者权益包方可调用                            |
| **ANi Open**        | 一个支持直链的番剧公益站，网络环境问题可能需要镜像才能使用         |

## 驱动补充说明

以下是特定驱动的一些帮助信息：

::: details 夸克网盘

20260212 起，受夸克 API 改动影响，仅填 Cookie 无法获取302直链。

如需使用转码直链(302)需要抓移动端任一 drive-m.../file 的请求链接。[抓取方法](/help/resource#夸克网盘移动端-url-抓包)

:::

::: details 天翼云盘

支持 `.cas` 秒传文件的直链解析：**STRM设置->媒体文件后缀** 中加入 cas 扩展名，可以直接播放 cas 生成的 strm 。

**.cas 支持特性与说明：**
- 通过家庭共享秒传临时文件获取直链，自动清理临时文件（含回收站）
- 直链已缓存优化，有效期内二次请求将直接返回直链，而非重复秒传
- 理论上媒体库视频编码可以正常获取，和正常入库视频格式 strm 无异
- 需注意大批量入库时神医并发扫视频编码，有可能会挤爆家庭云
- 为确保能顺利秒传，家庭云应留相应大小的空间

:::
