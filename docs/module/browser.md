# 存储浏览

SmartStrm 提供了一个 **存储浏览** 模块，允许你直接在 Web 界面中预览远端文件，方便直观地创建任务。

## 文件浏览器

在 **存储浏览** 标签页中，你可以：
- **切换存储**：在页面左上方的下拉菜单中选择已添加的存储驱动
- **目录导航**：支持点击文件夹进入、上一级返回，或直接在地址栏输入路径
- **基本操作**：右键点击文件或文件夹，可以进行 **重命名** 或 **删除** 操作
- **路径搜索**：在当前目录下快速过滤查找特定文件

## 批量命名

内置了三种重命名模式，帮助你快速整理杂乱的网盘文件名，使其符合媒体库识别规范。

### 1. 魔法变量
程序会自动尝试识别文件名中的标题、集数等信息。
- **命名模板**：你可以自定义命名格式，如 `S{season}E{episode}.{ext}`

### 2. 正则替换
利用正则表达式进行精准替换。
- **匹配**：输入正则表达式，如 `EP(\d+)`
- **替换**：输入替换后的内容，如 `E$1`

### 3. 顺序命名
针对没有规律的文件名进行纯体力活重命名。
- **配置项**：前缀（如 `S01E`）、起始编号、补位宽度（如 `2` 表示 `01, 02...`）、后缀。

## 影视识别

在文件浏览器中点击 **影视识别** 按钮，可以联动 TMDB API 对当前目录下的资源进行智能识别。

1. **选择类型**：电影、剧集或混合
2. **手动修正**：如果自动识别不准，可以手动输入名称或年份进行二次搜索
3. **海报预览**：实时展示识别到的影视海报、评分及简介
4. **一键重命名**：识别成功后，可以根据 TMDB 返回的各种元数据变量（如 `{tmdbid}`、`{episode_name}`）进行批量更名

> [!TIP] 使用前提
> 此功能依赖于你在 “系统设置->影视识别” 中配置的 TMDB API 密钥，推荐配置网络代理以确保稳定访问。
