# 快速开始

按照以下前四个步骤，可以通过 SmartStrm 让你的 Emby 媒体库快速充满内容。

## 第一步：添加存储

登录管理后台后，点击左侧菜单的 **存储管理**。

1. 点击 **添加存储**。
2. 选择你要使用的驱动（如 115、夸克网盘、OpenList 等）。
3. 根据驱动要求填写授权信息。
4. 保存并检查连接状态。

## 第二步：添加任务

点击左侧菜单的 **任务管理**。

1. 点击 **添加任务**。
   - **任务名称**：起一个好记的名字（如 `movie`）
   - **使用存储**：选择你在第一步中添加的存储
   - **扫描路径**：网盘中媒体文件存放的路径
2. 点击 **保存** 后，运行一次任务

## 第三步：共享 STRM 目录

将 SmartStrm 的 `/strm` 目录同时挂载给你的媒体服务器（Emby/Jellyfin/Plex），使其能够访问到 SmartStrm 生成的文件。

通常在部署 Docker 时，我们会将同一个宿主机目录挂载给两个容器。例如：

- **SmartStrm**: `/volume1/media/strm:/strm`
- **Emby**: `/volume1/media/strm:/strm`

## 第四步：媒体库扫描

在 Emby 的后台管理中：

1. 进入 **媒体库** -> **添加媒体库**
2. 文件夹选择你挂载后的路径（如 `/strm`）
3. 选择对应的内容类型（电影/电视节目）
4. 开启扫描

完成以上步骤，你就能在 Emby 中看到云端的视频并直接播放了！🥳

> [!INFO] 进阶优化
> 此时所有视频流量经过 Emby 服务端，如果你在局域网外访问，可能受限于家中网络的上传带宽（特别是多人同时播放时）。你可能还需要：**第五步：设置302代理**

## 第五步：302代理（Pro）

进入 SmartStrm **系统设置** -> **302代理**

1. **服务端口**：自定义代理后提供访问的端口
2. **媒体服务器地址**：填写 Emby 的访问地址
3. 保存设置和启用代理服务器
4. 在你的客户端中，改用 1 中填写的端口访问 Emby

这将使播放客户端点击播放时，直接从网盘直链获取视频流，享受最顺畅的播放。🎉
