# 存储驱动说明

SmartStrm 支持多种网络存储驱动，你可以根据自己的网盘账号类型进行选择。

## 支持的驱动列表

| 驱动名称 | 适用范围 | 直链支持 | 说明 |
| :--- | :--- | :--- | :--- |
| **OpenList** | 各类网盘、WebDAV | 取决于源盘 | 配合 [OpenList](https://github.com/Cp0204/OpenList) 项目使用 |
| **WebDAV** | ALis, CloudDrive2 等 | 取决于实现 | 标准 WebDAV 协议 |
| **本地存储** | 宿主机/容器本地路径 | N/A | 用于将本地挂载的盘映射为 strm |
| **夸克网盘** | 夸克账号 | 需 302 代理 | 支持生成原文件或转码流 strm |
| **Quark TV** | 夸克 TV 端协议 | ✅ | 相比网页版更稳定的直链获取方式 |
| **115 开放平台** | 115 网盘 | 完美 | 暂不支持 115 登录，需 115 扫码授权或 Cookie |
| **天翼云盘** | 天翼云盘 | 完美 | 速度极快，适合 4K |
| **123 云盘开放平台** | 123 云盘 | 完美 | 基于官方开放平台接口 |
| **ANi** | ANi Open 服务 | ✅ | 专门针对动漫资源的驱动 |

## 驱动配置要点

### OpenList
- 推荐开启 OpenList 的 302 功能。
- 如果 OpenList 挂载在 SmartStrm 相同的宿主机，建议使用内网 IP 进行连接。

### 115 开放平台
- 直链兼容性极好。
- 注意：115 开放平台普通用户的 API 调用频率有限制，建议避免过频繁的扫描。

### 夸克网盘
- 夸克的部分视频受限于平台转码，可能在某些客户端播放偏色。
- 支持 Pro 版的智能回落机制。

### 123 云盘
- 需要在 123 云盘开放平台申请。

---

> [!TIP]
>
> **提示**：在使用 302 直链播放功能时，务必确保驱动本身能提供持久且高速的直链。天翼云和 115 目前是较为理想的首选。
